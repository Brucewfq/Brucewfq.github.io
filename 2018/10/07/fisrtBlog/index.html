使用gitHub + Hexo搭建个人博客以及成功之后的一些问题

发表于 2018-10-07

搭建过程还是有点小问题，对于hexo还不是很熟悉，发博客什么的都不太了解，这个仅仅是记录一下怎么搭建，参照网上的教程
想要看具体安装可以通过这个传送门：https://blog.csdn.net/gdutxiaoxu/article/details/53576018

下面主要说说安装成功之后的一些问题

看到这里的朋友默认已经安装成功，在我们安装成功之后，肯定会想怎么来发布自己的博客，除了通过 $ hexo new “My New Post” 这个命令之外，我们
还可以在source/_posts这个目录下手动新建.md文件，新建成功之后使用 $ hexo d -g 命令来发布，不需要把项目push到gitHub上，至此就完成了一个博客。

开启评论功能：使用gitment
$npm install gitment –save #安装gitment

然后在 https://github.com/settings/applications/new 进行注册，获取Client ID和Client Secret
